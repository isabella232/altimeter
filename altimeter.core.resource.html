static merge_resources(resource_id, resources)¶

Merge multiple resources with the same id into one. This is permissible in two cases:

1) If all resources have the same value for ‘get_full_type’ and no key/values conflict they will be merged into a single resource by combining all key/values into a single resource

2) If all resources do not have the same value for ‘get_full_type’, if classes have the allow_clobber attribute set depending on the values a resource may be created.

Parameters

  • resource_id (str) – common resource id

  • resources (List[Resource]) – list of Resources to merge

Return type

Resource

Returns

merged Resource object

Raises

UnmergableDuplicateResourceIdsFoundException if resources could not be merged. –

altimeter.aws.scan.scan_manifest module¶

A ScanManifest defines the output of a complete scan.

class ScanManifest(scanned_accounts, master_artifact, artifacts, errors, unscanned_accounts, api_call_stats, start_time, end_time)¶

Bases: object

A ScanManifest defines the output of a complete scan. It contains pointers to the per-account scan result artifacts and summaries of what was scanned, errors which occurred, scan datetime and api call statistics.

Parameters

  • scanned_accounts (List[str]) – List of account ids which were scanned

  • master_artifact (str) – artifact containing complete graph json

  • artifacts (List[str]) – list of artifacts, one per account

  • errors (Dict[str, List[str]]) – Dict of account_ids to list of errors encountered during scan

  • unscanned_accounts (List[str]) – List of account ids which were not scanned

  • api_call_stats (Dict[str, Any]) – api call stats for this scan

  • start_time (int) – epoch timestamp of scan start time

  • end_time (int) – epoch timestamp of scan end time
